## Блок с текущим статусом резюме:
- Рекомендуемые для заполнения поля (максимум 5, остальные скрыты), порядок: - 1,5
  - Обязательные поля или ошибки модерации
  - Необязательные поля
- Статус публикации резюме - 1
  - Если заполнены все обязательные поля, дать возможность опубликовать
  - Если опубликовано и прошло модерацию показываем ссылку на настройку видимости
  - Если в модерации, пишет об этом сообщение
  - Если не прошло модерацию — выводим ошибки (в виде рекомендуемых полей)
- Тип видимости (ссылкой на настройку видимости) - 0,5
  - Видно всему интернету
  - Видно зарегистрированным на hh работодателям
  - Не видно никому
  - Видно по прямой ссылке
  - Никто, кроме белого списка
  - Все, кроме чёрного списка

- Не видно никому

## Образование
Селект с элементами справочника `education_level`:
  - Высшее
  - Бакалавр
  - Магистр
  - Кандидат наук
  - Доктор наук
  - Неоконченное высшее
  - Среднее
  - Среднее специальное

При выборе значения «Среднее» отображается форма указания среднего образования,
при выборе остальных — форма указания высшего образования.

### Среднее образование
  - поле ввода:
    - лейбл «Название учебного заведения»
    - ограничение `maxlength` получается через метод api `resume_conditions`
  - текстовое поле:
    - лейбл «Год окончания»
    - минимальное и максимальное значения получаются через метод api `resume_conditions`
    - подсказка «Если учитесь в настоящее время — укажите год предполагаемого окончания»
  - кнопка «×» для удаления записи, если эта запись — не единственная
  - ссылка для добавления новой записи «Указать еще одно место обучения»

### Высшее образование
  - поле ввода:
    - лейбл «Название учебного заведения»
    - плейсхолдер «Например, «Московский государственный университет им. М.В. Ломоносова»
    - подсказка «Введите название или аббревиатуру учебного заведения,
      и мы предложим выбрать из списка точное название»
    - валидируется заполнение поля, если поле не заполнено — поле красное,
      выводится предупреждающая иконка
    - ограничение `maxlength` получается через метод api `resume_conditions`

  - поле ввода:

### Владение языками (language)
  - Список языков с полем уровня
  - Каждый элемент списка имеет кнопку удаления
  - Под всеми элементами есть кнопка добавления дополнительного элемента в список
  - Элемент:
    - Дропдаун с выбором языка
      - В дропдауне отсутствуют уже добавленные языки
    - Дропдаун с уровнем языка
    - Кнопка удаления
  - Список языков можно получить по адресу https://api.hh.ru/languages
  - Список уровней языка можно получить в https://api.hh.ru/dictionaries в "language_level"

### Опыт работы (experience)
  - Список работ
  - Под всеми элементами есть кнопка добавления дополнительного элемента в список
  - Элемент:
    - Организация (company)
      - Текстовое поле
    - Регион расположения организации (area)
      - Текстовое поле с возможностью выбора обьекта из структуры https://api.hh.ru/areas
        и подсказками при вводе
    - Сайт компании (company_url)
      - Текстовое поле с проверкой корректности url
    - Отрасль (industry)
      - Дропдаун с элементами из https://api.hh.ru/industries
    - Должность (position)
      - Текстовое поле 
    - Начало работы (start)
      - Выбор месяца и года
    - По настоящее время
      - Чекбокс
    - Окончание работы (end)
      - Выбор месяца и года
      - Неактивно, если активен чекбокс "По настоящее время"
    - Обязанности и функции (description):
      - Текстовое поле
    -  Кнопка удаления

### Дополнительная информация (skills)
  - Текстовое поле для заполнения в произвольной форме

### Ключевые навыки (skill_set)
  - Список уникальных строк
  - Каждый элемент списка имеет кнопку удаления
  - Текстовое поле + кнопка добавления
  - Функциональность ввода всех навыков в одну строчку
    и автоматический парсинг по запятой
  - Есть отдельный саджест

### Список рекомендаций (recommendation)
  - Список
  - Под всеми элементами списка есть кнопка добавления дополнительного элемента в список
  - Элементы списка:
    - Имя (name)
      - Текстовое поле
    - Должность (position)
      - Текстовое поле
    - Организация (organization)
      - Текстовое поле
    - Телефон (contact)
      - Текстовое поле
      - При заполнении поля происходить автоматическое форматированние
    - Кнопка удаления
